# sql-startup-funding-analysis
SQL project for data cleaning and analysis

Summary:
This project focuses on analyzing startup funding data using SQL. The goal is to clean raw data, handle inconsistencies, and extract meaningful insights about funding trends across industries, cities, and investors.

Objectives:
•	Clean raw dataset
•	Handle missing values and remove duplicate records
•	Analyze funding distribution across different industries
•	Identify top cities and investors contributing to startup funding
•	Answer key business questions using SQL queries

Tools :
•	SQL (MySQL)
•	Data Cleaning Techniques
•	Aggregate Functions and Joins

Data Cleaning Steps:
•	Replaced or handled NULL values appropriately
•	Removed duplicate records to improve data accuracy
•	Standardized inconsistent data entries

Key Analysis Performed:
•	Industry-wise total funding analysis using GROUP BY
•	City-wise funding trends and comparisons
•	Identification of top-funded startups
•	Analysis of leading investors and their investment patterns
•	Use of JOIN operations to combine multiple datasets

 


Conclusion:
This project demonstrates practical use of SQL for data cleaning and analysis. It highlights the importance of structured querying, data accuracy, and extracting actionable insights from raw datasets.
(dataset used from kaggle)
